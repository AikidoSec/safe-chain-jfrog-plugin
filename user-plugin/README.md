# Artifactory Stop Remote Download Plugin

This plugin prevents downloads from remote repositories in JFrog Artifactory.

## Features
- Intercepts download requests
- Blocks downloads from remote repositories
- Logs blocked requests

## Installation
1. Place the `StopRemoteDownload.groovy` script in your Artifactory's `$ARTIFACTORY_HOME/etc/plugins` directory.
2. Restart Artifactory.

## Configuration
No additional configuration is required.
